# 📂 ReimbuS - Versi 1 (Legacy / Pre-Revision)

Repositori ini berisi kode sumber **versi pertama** dari aplikasi Reimbursement Karyawan (ReimbuS). Proyek ini dikembangkan selama masa magang di **PT Data Andalan Utama** sebagai tahap awal pengujian fungsionalitas sistem.

---

## 📝 Deskripsi Proyek
Versi ini merupakan *prototype* awal yang berfokus pada implementasi fitur dasar sistem klaim menggunakan **Payload CMS v3** dan **Next.js**. Kode di dalam repositori ini mencakup logika awal untuk pengajuan reimbursement, manajemen karyawan, dan integrasi database.

### 🛠️ Spesifikasi Teknis (Legacy)
* **Framework:** Next.js (App Router) & Payload CMS v3.
* **Styling:** Menggunakan **Plain CSS** (CSS biasa) dengan struktur penulisan gaya yang masih terpisah.
* **Arsitektur:** Menggunakan pola monolitik sederhana, di mana logika bisnis (*handlers*) dan tampilan UI masih berada dalam file yang sama (misal: `EditForm.tsx`).
* **Database:** Terkoneksi dengan **MongoDB** melalui Mongoose Adapter.
